If $U \subseteq X$ is an infinite and coinfinite set which does not contain $p$, then $U$ is open, however $\overline{U} = U \cup \{ p \}$ is not open.

See item #7 for space #24 in {{doi:10.1007/978-1-4612-6290-9_6}}.
