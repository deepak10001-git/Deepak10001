$\{p\}$ is closed. Let $U_n = X \setminus A_n$ be an open set containing $\{p\}$. Then $\bigcap_n U_n = X \setminus \bigcup_n A_n$ is uncountable, so $\{p\}$ is not $G_\delta$.
