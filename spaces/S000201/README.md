The subspace of {S176} defined by
$$X=\bigcup_{n=1}^{\infty}\left\{(x,y)\in\mathbb{R}^2\middle\mid\left(x-\frac{1}{n}\right)^2+y^2=\left(\frac{1}{n}\right)^2\right\}.$$

Defined as "infinite earring" in section 71 of {{zb:0951.54001}}.
