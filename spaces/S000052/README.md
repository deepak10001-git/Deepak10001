For any $a,b \in X = \mathbb{Z}^+$, let $U_a(b) = \{b+na : n \in \mathbb{Z}\}$. Then $\{U_a(b) : a,b \in X, gcd(a,b)=1\}$ forms a basis for the relatively prime integer topology on $X$.
