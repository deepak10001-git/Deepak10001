The Warsaw circle is the set $\{(x, \sin\frac{1}{x})\ |\ x \in (0,1]\} \cup \{(0,y)\ |\ y \in [-1,2]\} \cup \{(x,2)\ |\ x\in[0,1]\} \cup \{(1,x) \|\ x \in [\sin(1),2]\}$ with the subspace topology inherited from $\mathbb{R}^2$.
